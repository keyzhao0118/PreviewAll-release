# Preview All

**Preview All** 是一款专注于「本地文件快速预览」的桌面应用，致力于在**不联网、不上传、不收集任何用户数据**的前提下，为用户提供稳定、高效、可扩展的文件预览能力。

---

## ✨ 特性概览

- 📂 **多格式文件预览**  
  支持对多种常见及专业文件格式进行快速预览，无需打开原始应用

- ⚡ **高性能本地解析**  
  所有文件内容仅在本地解析与渲染，不依赖云服务

- 🔒 **隐私优先设计**  
  - 不联网  
  - 不上传文件  
  - 不收集任何用户数据  
  - 不包含任何分析或统计代码  

- 🧩 **模块化架构**  
  预览能力按格式解耦，便于后续扩展新的文件类型

- 🪟 **原生桌面体验**  
  专为 Windows 桌面环境设计，注重响应速度与稳定性

---

## 🖼 使用场景

Preview All 适用于但不限于以下场景：

- 快速查看文件内容而不打断当前工作流  
- 在文件管理、归档、分析过程中进行只读预览  
- 对不信任来源的文件进行**安全、离线查看**  
- 开发者、设计师、工程人员的辅助工具  

---

## 🔐 隐私与安全

Preview All 从设计之初就将**隐私保护**作为核心原则：

- 所有文件 **仅在本地处理**
- 应用 **不会进行任何网络连接**
- 不会记录、存储或上传文件内容、文件路径或用户行为
- 不使用任何在线服务或数据分析 SDK

👉 详细内容请参阅：隐私策略（Privacy Policy）

---

## 🛠 技术说明（简要）

- 使用现代 C++ 与 Qt 构建
- 依赖部分开源第三方库（通过 vcpkg 管理）
- 所有第三方组件均在本地运行，不涉及网络或数据收集
- 注重稳定性、可维护性与长期扩展能力
